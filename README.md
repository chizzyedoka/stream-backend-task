## Description

This project is a simple NestJS API for handling user transactions with MongoDB (STREAM Backend Intern Task). It supports creating users, crediting and debiting their wallet balances, and ensures that balance updates are accurate and secure.

## Features

- **User Management**: Create users with hashed passwords.
- **Transaction Management**: Handle debit and credit transactions, updating user wallet balances accordingly.
- **Validation**: Ensures proper data validation using DTOs.
- **Testing**: Includes unit tests for the service logic and integration tests for API endpoints.

## Getting Started

### 1. Clone the Repository

```bash
$ git clone https://github.com/chizzyedoka/stream-backend-task

$ cd stream-backend-task
```

### 2. Install Dependencies

```bash
$ npm install
```

### 3. Set Up MongoDB

Go to the app.module.ts file and insert the necessary imports

```bash
 $ imports: [
    MongooseModule.forRoot('mongodb://localhost/stream')
  ]
```

- Replace 'mongodb://localhost/stream' with your MongoDB connection string if different.

### 4. Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### 5. Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

```

### 6. API Endpoints

```bash
# Create User
$ POST /users

#Request Body:

#json
$ {"username": "testuser","email": "test@example.com","password": "password123" }

# Create Transaction
$ POST /transactions

#Request Body:

#json
$ {"amount": 100,"type": "credit","userId": "userId123" }
```
